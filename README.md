# Fumi’s Hair Haven 💎

This website was designed and developed for Fumi’s Hair Haven, a student-run wig service based in Hatfield, Pretoria.

## ✨ About This Project

This project was created to support a small, student-owned business by giving it a clean and professional online presence.

The goal was to design a website that feels simple, trustworthy, and well put together — something that makes it easy for clients to explore services and book with confidence.

## 💭 My Approach

When building this website, I focused on:

* Keeping the layout clean and easy to navigate
* Making the booking process straightforward
* Creating a design that feels polished but not overwhelming
* Ensuring it works smoothly on both mobile and desktop

Even though it’s a small business, I wanted the website to feel intentional and well-designed.

## 🖥️ Features

* Responsive design (mobile + desktop)
* Smooth scrolling navigation
* Service section with clear structure
* Booking form with dynamic pricing
* WhatsApp integration for bookings
* Image gallery to showcase work
* Policies and payment information section

## 🎨 Design Style

The design is minimal and modern, with:

* Neutral tones (cream, white, charcoal)
* Subtle gold accents
* Elegant typography (Playfair Display & Poppins)
* Clean spacing for readability
